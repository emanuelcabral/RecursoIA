# 📘 RecursoIA

Repositorio que reúne **recursos y herramientas de Inteligencia Artificial (IA)**.  
Su propósito es ofrecer un directorio centralizado y accesible para explorar aplicaciones de IA en distintas áreas.

---

## 🚀 Categorías principales
- **Chats conversacionales**: Plataformas de diálogo con modelos de lenguaje.  
- **Asistentes para documentos**: IA aplicada a la lectura, análisis y resumen de archivos.  
- **Generadores de imágenes**: Creación de imágenes a partir de texto.  
- **Edición de imágenes**: Herramientas para mejorar, transformar o limpiar fotografías.  
- **Texto a voz (TTS)**: Conversión de texto en audio con voces naturales.  
- **Creadores de música**: Producción musical asistida por IA.  
- **Generación de videos**: Creación de clips y animaciones mediante modelos generativos.  
- **Generación de sitios web**: Constructores web potenciados por IA.  
- **Productividad**: Aplicaciones que integran IA para optimizar el trabajo diario.
